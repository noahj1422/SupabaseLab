# SupaBaseLab

A basic Node.js web application with Express.js

## Features

- Node.js server with Express.js framework
- Responsive main page with modern design
- Static file serving
- Easy to extend and customize

## Installation

1. Make sure you have Node.js installed (v14 or higher)
2. Install dependencies:

```bash
npm install
```

## Running the Server

Start the development server:

```bash
npm start
```

The application will be available at `http://localhost:3000`

## Project Structure

```
supaBaseLab/
├── main.js              # Express server configuration
├── package.json         # Node.js project configuration
├── public/
│   └── index.html       # Main page
└── README.md            # This file
```

## Customization

- Edit `public/index.html` to customize the main page
- Add more routes in `main.js` as needed
- Install additional packages via `npm install <package-name>`
